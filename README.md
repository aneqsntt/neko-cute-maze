# 🐱 Neko Maze Run: Retro Edition

Bem-vindo ao **Neko Maze Run**, um jogo de labirinto no estilo pixel art, super fofo e com uma pegada retrô, inspirado nos clássicos dos portáteis de antigamente!

## 🎮 Sobre o Jogo
O objetivo é simples: guie o gatinho até a estrela para avançar de nível. Mas cuidado! O caminho é cheio de paredes e o desafio aumenta a cada um dos **10 níveis** disponíveis.

### Funcionalidades:
* **Interface Retrô:** Estética nostálgica com cores pastéis.
* **10 Níveis Progressivos:** O labirinto fica mais complexo conforme você avança.
* **Sistema de Vidas:** Você tem 3 corações. Se tocar nas paredes, cuidado com o choque!
* **Controles Intuitivos:** Compatível com teclado (Setas ou WASD) e botões na tela.

## 🚀 Como Jogar
1. Baixe ou clone este repositório para o seu computador.
2. Abra a pasta onde você salvou os arquivos.
3. Clique duas vezes no arquivo `index.html`.
4. O jogo abrirá automaticamente no seu navegador padrão.
5. Clique em **"START GAME"** e divirta-se!

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura do jogo.
* **CSS3:** Estilização e animações *pixel art*.
* **JavaScript:** Lógica de movimento, geração de labirintos e sistema de níveis.

## 📝 Como contribuir
Este é um projeto de aprendizado. Sinta-se à vontade para fazer um *fork*, alterar as cores, adicionar novos efeitos sonoros ou criar novos níveis!
